# 📚 纯前端文本书签导航工具

一个完全运行在浏览器中的本地书签管理工具，无需后端服务器，数据存储在浏览器的 `localStorage` 中，双击 HTML 文件即可使用。

## ✨ 功能特性

### 📋 核心功能

- **纯前端实现**：单 HTML 文件，无需安装，双击即可运行
- **数据本地存储**：所有数据保存在浏览器 localStorage 中
- **拖拽排序**：支持拖拽调整书签顺序，实时保存
- **智能搜索**：支持标题、备注、链接、标签、分类等多字段搜索
- **分页浏览**：每页12个书签（4列×3行），支持上一页/下一页导航
- **分类筛选**：按分类快速筛选书签，支持自定义分类管理

### 🏷️ 标签与分类

- **彩色标签**：预设彩色标签（爆/荐/新），支持自定义标签颜色
- **分类管理**：默认分类"常用"，支持添加、编辑、删除分类
- **智能关联**：删除分类或标签时检查关联书签，防止数据丢失

### 📁 数据管理

- **Excel 导入/导出**：支持导出为 `.xlsx` 文件，可导入追加或覆盖现有数据
- **批量导入**：支持从 Excel 文件批量导入书签，自动处理分类和标签
- **数据备份**：导出功能可用于定期备份书签数据

### 🎨 界面与交互

- **图片预览**：鼠标悬停在书签标题上显示图片预览
- **备注提示**：显示书签备注，悬停显示完整内容
- **响应式设计**：适配桌面和移动设备
- **简洁界面**：无多余标题，专注于功能操作

## 🚀 快速开始

### 使用方法

1. **直接运行**：双击 `bookmark-navigator.html` 文件在浏览器中打开
2. **添加书签**：点击工具栏中的"➕ 添加书签"按钮
3. **分类筛选**：点击分类筛选栏中的分类名称
4. **搜索书签**：使用搜索框输入关键词，支持回车搜索
5. **拖拽排序**：按住书签项拖拽到其他位置交换顺序

### 首次使用

工具首次打开时会自动加载示例数据：

- 3个示例书签（GitHub、MDN、Google）
- 3个默认分类（常用、编程、设计）
- 3个预设标签（爆、荐、新）

## ⚙️ 技术细节

### 技术栈

- **前端**：纯 HTML5 + CSS3 + 原生 JavaScript (ES6)
- **布局**：CSS Grid 实现响应式网格布局
- **存储**：浏览器 localStorage 数据持久化
- **外部库**：
  - [SheetJS](https://cdn.sheetjs.com/) - Excel 文件处理
  - [Font Awesome 6](https://cdnjs.cloudflare.com) - 图标字体

### 数据模型

```javascript
// 书签对象
{
  id: "唯一标识",
  title: "书签标题",
  url: "链接地址",
  imageUrl: "图片预览链接",
  note: "备注信息",
  category: "分类名称",
  tag: "标签名称"
}

// 标签对象
{
  name: "标签名称",
  color: "#十六进制颜色"
}
```

### 存储结构

```javascript
// localStorage 键名
bookmark_navigator_bookmarks    // 书签列表
bookmark_navigator_categories   // 分类列表
bookmark_navigator_tags         // 标签列表
```

## 📖 详细功能说明

### 1. 添加书签

- 弹出表单包含：标题（必填）、链接（必填）、图片URL、备注、分类（下拉选择）、标签（下拉选择）
- 支持在分类和标签下拉框中输入新建分类/标签
- 图片URL用于鼠标悬停预览

### 2. 搜索功能

- **多字段搜索**：同时在标题、备注、链接、标签、分类中搜索
- **即时反馈**：搜索结果实时更新
- **清除搜索**：一键清除搜索条件

### 3. 分类管理

- **默认分类**："常用"分类不可删除
- **自定义分类**：支持添加新分类，删除前检查关联书签
- **分类筛选**：点击分类按钮筛选对应书签

### 4. 标签管理

- **颜色标签**：每个标签有名称和颜色
- **预设标签**：爆（红色）、荐（绿色）、新（蓝色）
- **自定义标签**：支持添加新标签，选择颜色
- **标签编辑**：可修改标签名称和颜色

### 5. 导入/导出

- **导出 Excel**：生成包含所有书签信息的 `.xlsx` 文件
- **导入 Excel**：支持追加导入或覆盖导入
- **智能处理**：导入时自动处理新分类和新标签
- **数据验证**：跳过无效数据行，提示导入结果

### 6. 分页系统

- **每页12个**：4列×3行的网格布局
- **自动调整**：删除书签后自动调整页码
- **状态显示**：显示当前页码和总页数

### 7. 拖拽排序

- **直观操作**：拖拽书签项到其他位置
- **视觉反馈**：拖拽时有半透明效果和虚线边框
- **即时保存**：顺序变化立即保存到 localStorage

## 🔧 兼容性

- **浏览器**：支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- **运行环境**：支持 `file://` 协议，无需HTTP服务器
- **设备**：支持桌面和移动设备，响应式布局

## 💡 使用建议

### 数据备份

1. 定期使用"导出 Excel"功能备份书签数据
2. 备份文件可存储在其他位置或云端
3. 需要恢复时使用"导入 Excel"功能

### 分类策略

1. 建议按使用场景创建分类（工作、学习、娱乐等）
2. 使用标签进行更细粒度的标记
3. 合理控制分类和标签数量，避免过多

### 图片预览

1. 建议使用尺寸适中的图片（最大200×200px）
2. 使用可靠的图床或CDN托管图片
3. 图片加载失败时有优雅的降级处理

## 🐛 故障排除

### 常见问题

1. **数据丢失**：检查浏览器是否启用了 localStorage
2. **导入失败**：确保 Excel 文件格式正确（包含标题行和6列）
3. **拖拽无效**：确保浏览器支持 HTML5 拖放 API
4. **图片不显示**：检查图片URL是否有效，网络是否正常

### 数据恢复

1. 如果误删书签，可重新导入最近的备份文件
2. 分类和标签删除有保护机制，需先处理关联书签
3. 重要数据建议定期导出备份
